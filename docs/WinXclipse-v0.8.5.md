# WinXclipse v0.8.5 - Driver Delivery Manual

## Objetivo

Adicionar uma etapa inicial de entrega de drivers/wrappers externos para o WinXclipse, usando a release da MdiEx v3.2 como fonte de pacotes compatíveis.

## Fonte dos drivers

Os drivers e wrappers externos podem ser baixados pela release:

https://github.com/avavo/MdiEx/releases/tag/V3.2

## Instalação manual

1. Baixe o pacote desejado pela release da MdiEx v3.2.
2. Abra o WinXclipse.
3. Vá em Xclipse GPU Drivers.
4. Toque em Install Driver.
5. Selecione o arquivo baixado.
6. Reinicie ou recrie o container se necessário.

## Observação

Nem todo driver funciona em todo aparelho, emulador ou versão do Android. Use apenas pacotes compatíveis com Xclipse/WinXclipse.
