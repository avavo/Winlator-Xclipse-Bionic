# WinXclipse v0.8.5 - Driver Delivery Manual

## Objetivo

Adicionar um fluxo manual mais claro para instalação de drivers/wrappers externos no WinXclipse.

## Mudanças

- Adicionado link para a release MdiEx v3.2 na tela Xclipse GPU Drivers
- O botão Install Drivers continua abrindo o seletor de arquivos
- O usuário pode baixar drivers compatíveis pela release e instalar manualmente pelo app

## Fonte dos drivers

https://github.com/avavo/MdiEx/releases/tag/V3.2

## Instalação manual

1. Abra Xclipse GPU Drivers no WinXclipse.
2. Toque no link da MdiEx v3.2.
3. Baixe o pacote desejado.
4. Volte ao WinXclipse.
5. Toque em Install Drivers.
6. Selecione o arquivo baixado.

## Próxima etapa

Para v0.9.0, será estudado um botão de download direto pelo app, com lista remota de drivers e instalação automática.
