# WinXclipse v0.8.0 - Runtime + Identity Base

## Objetivo

Tornar o WinXclipse mais independente da base Cmod, com package próprio, identidade visual corrigida e entrega inicial de runtimes via GitHub Releases.

## Mudanças

- Package name alterado de \`com.winlator.cmod\` para \`com.win.xclipse\`
- WinXclipse pode ser instalado lado a lado com Cmod
- Nome do app corrigido para \`WinXclipse\`
- Branding, ícones e About corrigidos
- Textos de instalação de driver atualizados para WinXclipse/Xclipse
- Removidas menções antigas a Qualcomm/Adreno no aviso de instalação
- FEXCore 2601-aabdded e 2605 adicionados via GitHub Releases
- Wine/Proton 10 arm64ec e 11.0-1 arm64ec adicionados via GitHub Releases
- Download e instalação dos runtimes testados com sucesso
